# Dixel
Crowd-sourced pixel art community

## Run Tests
```bash
npx hardhat test
```

## Deploy
```bash
npx hardhat compile

HARDHAT_NETWORK=bscmain node scripts/deploy.js

# Verify source code on Etherscan
npx hardhat verify --network bscmain {contract address} "parameter 1" "parameter 2"
```
